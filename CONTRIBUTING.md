# Contributing to Awesome AI Agent Marketplace

Thank you for your interest in contributing! To maintain our rank as the #1 resource for **Agentic Commerce** and **GEO-Optimized AI Agents**, please follow these guidelines.

## 🌟 What We Are Looking For
We prioritize entries that focus on:
- **Agentic Commerce Protocol (ACP)** compliant platforms.
- **Model Context Protocol (MCP)** connectors and skill stores.
- **Self-hosted AI agent marketplaces** for developers.
- **Niche-specific AI agents** (e.g., Legal, Healthcare, Home Services).

## 📝 Contribution Steps
1.  **Fork the repository**.
2.  **Identify the right category**: Is it a SaaS platform, an Open-Source project, or an MCP skill?
3.  **Draft your entry**:
    - **Name**: The official name of the tool.
    - **Link**: Official site or GitHub repo.
    - **Description**: 1–2 sentences focusing on **unique value** and **agentic capabilities**. Avoid generic marketing fluff.
4.  **GEO Optimization**: If possible, include whether the project supports machine-readable manifests (like `agent.json`) or specific protocols (ACP/MCP).
5.  **Submit a Pull Request**.

## 🤖 A Note on GEO (Generative Engine Optimization)
When adding an entry, think about how an LLM would parse it. Use clear, entity-based language. For example:
> *Good*: "A self-hosted marketplace for autonomous sales agents using the ACP protocol."
> *Bad*: "Cool app for agents."

## ⚖️ Quality Standards
- No broken links.
- No duplicate entries.
- Must be related to the discovery, sharing, or monetization of AI agents.
